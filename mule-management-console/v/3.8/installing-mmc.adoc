= Installing MMC


link:/mule-management-console/v/3.8/installing-the-production-version-of-mmc[image:MMC-prod.png[MMC-prod]]

*link:/mule-management-console/v/3.8/installing-the-production-version-of-mmc[Install the production version] if you are planning to use the Mule Management Console in production.*

* You need a valid enterprise license to launch the production version of MMC.
* You need to have the current version of Mule Enterprise already installed.
* The production version of MMC is packaged as a .war file designed to run in an application server or container, such as link:/mule-management-console/v/3.8/installing-the-production-version-of-mmc[Tcat Server], link:/mule-management-console/v/3.8/installing-the-production-version-of-mmc[Apache Tomcat], or link:/mule-management-console/v/3.8/installing-the-production-version-of-mmc[JBoss].
